# PhotSat SST Analysis

This repository contains the main simulation files, processed datasets, and data-processing code used in the Master's Thesis **"Simulating Space-based Observations of Orbital Debris with PhotSat"**.

The project investigates the potential contribution of PhotSat, a 16U CubeSat designed for all-sky photometric monitoring, to Space Surveillance and Tracking (SST) applications as a secondary product of its nominal scientific operations.

## Repository contents

- `configuration/`: Main configuration files used for the SST simulations.
- `inputs/`: Input files used for the PhotSat and TFRM simulations.
- `outputs/`: Relevant simulation outputs used in the analysis.
- `data_processing/`: Jupyter notebooks used for data processing, detectability analysis, orbital classification, and comparison between PhotSat and TFRM observations.
- `processed_data/`: Processed datasets generated from the simulation outputs and used to obtain the results presented in the thesis.

## Simulation period

The simulations cover a three-day interval from 21 to 24 June 2025.

## Software

The SST simulations were performed using software developed by GMV. The software itself is not included in this repository. Only the simulation files and data relevant to the analysis presented in the thesis are provided.
