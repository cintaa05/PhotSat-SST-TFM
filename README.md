# PhotSat SST Analysis

This repository contains the main simulation files and data-processing code used in the Master's Thesis **"Simulating Space-based Observations of Orbital Debris with PhotSat"**.

The project investigates the potential contribution of PhotSat, a 16U CubeSat designed for all-sky photometric monitoring, to Space Surveillance and Tracking (SST) applications as a secondary product of its nominal scientific operations.

## Repository contents

- `simulation_files/`: Main input and configuration files used for the PhotSat and TFRM simulations.
- `data/`: Relevant simulation outputs and processed datasets used in the analysis.
- `notebooks/`: Jupyter notebooks used for data processing, detectability analysis, orbital classification, and comparison of PhotSat and TFRM observations.

## Simulation period

The simulations cover a three-day interval from 21 to 24 June 2025.

## Software

The SST simulations were performed using software developed by GMV. The software itself is not included in this repository. Only the files and processed data relevant to the analysis presented in the thesis are provided.
